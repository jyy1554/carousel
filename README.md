# carousel

## Day 1
페이지 로딩 시, 첫 번째 이미지 화면에 보이기
좌측, 우측 화살표 두개 보이기
화면 아래 쪽에 Dot 다섯개 보이기

##Day 2
좌측 화살표 클릭시 이전 이미지 보여주기
우측 화살표 클릭시 다음 이미지 보여주기

##Day 3
5번째 이미지에서 우측 화살표를 누를 경우, 1번째 이미지 보여주기
1번째 이미지에서 좌측 화살표를 누를 경우, 5번째 이미지 보여주기
이미지 하단의 Dot를 누를 경우, 해당 순번의 이미지 보여주기
